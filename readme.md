# :pencil2:  Парсер данных с помощью Selenium
___
## :file_folder: Структура проекта:

Проект будет состоять из парсеров двух сайтов, один из которых уже залит.
Суть заключается в том, что сайты будут отличаться по сложности работы с ними, но структура
у них будет такая же, поэтому я и решил объединить два парсера в один проект.

### :one: Парсер первого сайта [QuotesToScrape](https://quotes.toscrape.com/js/)
+ Лежит в папке Quotes
+ файл parser_quotes.py собирает нужную информацию с сайта
+ файл data_base.py эту информацию переводит в базу данных
+ файл show_database.py показывает работу с базой данных

### :two: Второй сайт - Предполагаются [Auto.ru](https://auto.ru/) или другие варианты
